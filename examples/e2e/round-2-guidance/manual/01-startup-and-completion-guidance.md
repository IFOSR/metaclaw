# Round 2 Manual Scenario 01: Startup And Completion Guidance

目标：验证用户启动 Metaclaw 后能立即看到任务盘面；完成一个任务后，系统会主动建议下一步。

## 前置条件

启动真实 TUI：

```bash
./metaclaw.sh start
```

## 输入步骤

先创建两个任务：

```text
整理 Phoenix 项目的周报，输出一个简短结论
```

```text
给张总写一封 Phoenix 项目进展邮件
```

等待第一个任务完成。

## 预期 TUI 展示

启动时应看到：

- 活跃任务数量
- blocked 数量
- 建议优先任务
- 至少一条建议原因

第一个任务完成后应看到：

- 一个结构化的“下一步建议”块
- 推荐任务 ID 或标题
- 为什么推荐这个任务

## 通过标准

- 用户一启动就知道当前该做什么
- 完成任务后不需要额外追问，系统会主动给出下一个动作
